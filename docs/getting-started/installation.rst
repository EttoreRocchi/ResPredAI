Installation
============

Install from PyPI:

.. code-block:: bash

    pip install respredai

Or install from source:

.. code-block:: bash

    git clone https://github.com/EttoreRocchi/ResPredAI.git
    cd ResPredAI
    # For development (includes pytest)
    pip install -e ".[dev]"

Testing the Installation
------------------------

Verify the installation:

.. code-block:: bash

    respredai --version
